# 4D 领导力测评 · 微信小程序

基于 GitHub [vibe-coding-compass-86](https://github.com/qtcao79-del/vibe-coding-compass-86) 重构的微信小程序版本。

## 核心功能

- **24 道强迫选择题** — 每题从绿/黄/蓝/橙四个选项中选一个最贴近的
- **四色能量矩阵** — 2x2 水位填充矩阵，直观展示你的四维得分
- **完整 AMBR 报告** — 包含大白话画像、底层驱动、高光表现、压力盲点、破局点、AMBR 跃迁路径、职场拼图、使用说明书
- **断点续答** — 答题中途退出自动保存进度
- **历史记录** — 每次测评结果自动保存，可随时回看

## 四色特质

| 颜色 | 类型 | 荣格 | 关键词 |
|------|------|------|--------|
| 🟢 绿色 | 培养型 | NF | 情感粘合剂、灯塔捍卫者 |
| 🟡 黄色 | 包容型 | SF | 无差别连接者、氛围营造大师 |
| 🔵 蓝色 | 展望型 | NT | 破局远见者、逻辑异教徒 |
| 🟠 橙色 | 指导型 | ST | 铁血捍卫者、秩序守护神 |

## 导入微信开发者工具

1. 打开微信开发者工具
2. 选择「导入项目」
3. 目录选择本文件夹
4. AppID 使用 `wxc690beba688b1732` 或你自己的
5. 点击导入即可

## 项目结构

```
├── app.js / app.json / app.wxss     # 应用入口
├── utils/
│   ├── questions.js                  # 24道强迫选择题
│   ├── content-db.js                 # 四色特质内容数据库
│   ├── scoring.js                    # 计分逻辑
│   └── storage.js                    # 本地存储
├── components/
│   └── color-matrix/                 # 2x2 水位矩阵组件
├── pages/
│   ├── index/                        # 首页（品牌 + 四色预览）
│   ├── assessment/                   # 答题页
│   ├── result/                       # 结果报告页
│   ├── history/                      # 历史记录
│   └── mine/                         # 我的
└── images/                           # 图标资源
```

Apple 极简主义 + 四色能量系统，浅色背景 (#F5F5F7)，大圆角卡片，水位动画，皇冠浮动效果。
